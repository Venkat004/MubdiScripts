=============
Mubdi Scripts
=============

This is a collection of a number of scripts that I've put together
over the years to take care of various little issues or problems. 
Many of these scripts are related to Astronomy, but many are just
general use scripts that I don't want to re-write every time I need
one. Documentation will come as scripts are added to this module.

**Warning:** I cannot guarantee that all of these scripts are error
free, nor that they work in all cases.
